💧 Water Quality Prediction System

An intelligent Machine Learning-based application that predicts whether water is Safe or Unsafe for drinking using multiple chemical and physical water quality parameters.
The system also provides a confidence score, detects faulty parameters, generates smart recommendations, and visualizes important data for better analysis.

🚀 Features
✅ Water Quality Prediction (Safe / Unsafe)
📊 Prediction Confidence Score
⚠️ Faulty Parameter Detection
💡 Smart Recommendations
📉 Data Visualization
🚨 Risk Level Indicator
🧪 Real-Time Parameter Analysis
⚡ Lightweight & Efficient ML System

🧠 How It Works
User enters water quality parameter values.
Input data is processed using NumPy.
Processed data is passed to a trained Machine Learning model.
The model predicts:
Water Quality Status
Confidence Score
The system then:
Detects faulty parameters
Generates smart recommendations
Displays risk level
Creates visualization charts

🧪 Water Quality Parameters

The system analyzes the following parameters:

Parameter	Range
pH	0 – 14
Hardness	0 – 500
Total Dissolved Solids (TDS)	0 – 2000
Chloramines	0 – 10
Sulfate	0 – 500
Conductivity	0 – 1000
Organic Carbon	0 – 20
Trihalomethanes	0 – 150
Turbidity	0 – 10
📊 Output

The system provides:

🔹 Water Quality Status: SAFE ✅ / UNSAFE ❌
🔹 Prediction Confidence Score
🔹 Faulty Parameter Detection
🔹 Smart Recommendations
🔹 Risk Level Indicator
🔹 Graphical Visualization
📉 Data Visualization

The project includes graphical analysis for important parameters such as:

📌 pH Level
📌 TDS
📌 Turbidity

A safety threshold reference line is also displayed for easy understanding of water quality conditions.
