💧 Water Quality Prediction System

An intelligent Machine Learning-based Water Quality Prediction System that analyzes multiple chemical and physical water quality parameters to determine whether water is Safe ✅ or Unsafe ❌ for drinking.

The system not only predicts water quality but also provides:

📊 Confidence Score
⚠️ Faulty Parameter Detection
💡 Smart Recommendations
🚨 Risk Level Analysis
📉 Data Visualization

Built using Python, NumPy, Matplotlib, and Scikit-learn, this lightweight yet powerful system helps users understand water conditions through real-time analysis and graphical insights.

🚀 Features

✅ Water Quality Prediction
Predicts whether water is safe or unsafe for drinking.

📊 Prediction Confidence Score
Displays the confidence level of the machine learning model.

⚠️ Faulty Parameter Detection
Identifies which water parameters are outside the safe range.

💡 Smart Recommendations
Provides suggestions to improve water quality.

🚨 Risk Level Indicator
Classifies water condition into different risk categories.

📉 Data Visualization
Visual representation of important parameters for better understanding.

🧪 Real-Time Parameter Analysis
Analyzes user-input water quality values instantly.

⚡ Lightweight & Efficient ML Model
Fast prediction system with optimized machine learning performance.

🧠 How It Works

The system follows a simple workflow:

1️⃣ User enters water quality parameter values
2️⃣ Input data is processed using NumPy
3️⃣ Processed data is passed to the trained Machine Learning Model
4️⃣ The model predicts:

Water Quality Status
Confidence Score

5️⃣ The system then:

Detects faulty parameters
Generates smart recommendations
Displays risk level
Creates graphical visualizations
🧪 Water Quality Parameters

The model analyzes the following parameters:

Parameter	Range
pH	0 – 14
Hardness	0 – 500
Total Dissolved Solids (TDS)	0 – 2000
Chloramines	0 – 10
Sulfate	0 – 500
Conductivity	0 – 1000
Organic Carbon	0 – 20
Trihalomethanes	0 – 150
Turbidity	0 – 10
